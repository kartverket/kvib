# Visually Hidden

Visually Hidden brukes til å skjule innhold fra den visuelle klienten, men holde det lesbart for skjermlesere.

```jsx
import { VisuallyHidden, VisuallyHiddenInput } from "@kvib/react";
```

<DocsContainer>
**Egnet til:**
- Å skjule innhold på siden som skal være lesbart for folk som bruker skjermlesere.
- Å gi informasjon til skjermlesere der man ikke vil eller kan vise denne informasjonen til alle brukere.

**Uegnet til:**

- Å skjule interaktivt innhold
