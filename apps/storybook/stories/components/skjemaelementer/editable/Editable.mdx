# Editable

EditableText brukes til å endre på tekst inline. Teksten ser ut som normal tekst, men endres til å bli et text input field når brukeren klikker eller fokuserer på det.

## Ta i bruk

```jsx
import { Editable, EditableInput, EditableTextarea, EditablePreview } from "@kvib/react";
```
