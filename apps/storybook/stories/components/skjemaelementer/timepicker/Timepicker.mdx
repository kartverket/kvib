# Timepicker

Timepicker lar brukeren velge et tidspunkt på en brukervennlig måte.

```jsx
import { Timepicker } from "@kvib/react";
```

<DocsContainer>

**Egnet til:**

- Velge tidspunkt
- Skrive inn tidspunkt
- Velge tidspunkt med klikk eller tastatur

**Uegnet til:**

- Om brukeren ikke kan endre tidspunkt selv burde den ikke brukes

<Feedback component="Timepicker" />

### Label

Timepicker skal alltid ha en label. Dersom Input av en eller annen grunn ikke har label må en `aria-label` legges til for å tilfredsstille tilgjengelighetskrav.
