# NumberInput

NumberInput ligner på Input-komponenten, men den har kontroller for å øke eller redusere numeriske verdier.

```jsx
import {
  NumberInput,
  NumberInputField,
  NumberInputStepper,
  NumberIncrementStepper,
  NumberDecrementStepper,
} from "@kvib/react";
```

**Egnet til:**

- Svar
- Tall med eller uten interaktive knapper

**Uegnet til:**

- Søk (bruk heller `Search` eller `SearchAsync`)
- Interaktiv datepicker (bruk heller `DatePicker`)
