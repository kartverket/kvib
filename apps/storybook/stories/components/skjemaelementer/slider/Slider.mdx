# Slider

Her er en oversikt over Slider-komponenten, og hvordan denne bygges opp av flere andre komponenter.

<Alert status="warning">
  <AlertIcon />
  En Slider skal alltid stå sammen med en Label. Dersom det mot formodning avviker fra dette må aria-label benyttes.
</Alert>

<br />

## Ta i bruk

```jsx
import { Slider, SliderTrack, SliderFilledTrack, SliderThumb, SliderMark } from "@kvib/react";
```
