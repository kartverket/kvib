# Select

Select er et komponent som gjør at brukere kan velge én verdi fra forhåndsdefinerte alternativer.

```jsx
import { Select } from "@kvib/react";
```

<DocsContainer>

### Antall alternativer

Select funker godt når det er 7 eller flere alternativer. Om det er færre alternativer vil det være enklere for brukerne å forholde seg til `Radio`. Vurder også om det er behov for å søke i eller filtrere alternativene. Da anbefales det å bruke Combobox.

### Label

Select skal alltid ha en label. Dersom Input av en eller annen grunn ikke har label må en aria-label legges til for å tilfredsstille tilgjengelighetskrav.
