# Tooltip

<Alert status="info">
  <AlertIcon />
  Kjent bug for Tooltip: `placement="auto-end"` legger seg ikke som ønsket.
</Alert>

Et tooltip er en informativ melding som dukker opp når en bruker samhandler med et element.

## Ta i bruk

```jsx
import { Tooltip } from "@kvib/react";
```

    <Tag ref={ref} {...rest}>
      {children}
    </Tag>
  </Box>
));

const CustomToolTip = () => (

<Tooltip label="Hover me">
  <CustomCard>Tag Here</CustomCard>
</Tooltip>
);

render(<CustomToolTip />);`} dark />
