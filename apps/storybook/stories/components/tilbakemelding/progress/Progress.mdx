# Progress

Progress brukes til å vise frem status på fremdrift for en oppgave som tar tid eller består av flere steg.

```jsx
import { Progress } from "@kvib/react";
```

**Egnet til:**
- Vise framgang i en prosess

**Uegnet til:**
- Vise lasting som nesten er umiddelbar
- vise lasting for prosesser som ikke er så viktige for brukeropplevelsen

    ### Progress vs. Spinner

    Progress burde velges over spinner hvis man kan se prosentvis framgang, som f.eks. upload/download av filer, eller skjema med flere steg
    (Der kan stepper vurderes også). Hvis ventetiden er minimal, burde man ta spinner istedenfor.
