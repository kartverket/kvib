# Avatar

Avatar brukes til å representere brukere. Den viser frem profilbilde, initialer eller fallback-ikon.

## Ta i bruk

```jsx
import { Avatar, AvatarAvatar, AvatarGroup } from "@kvib/react";
```

## Fallback Avatar

Det er to fallbacks dersom det er en error når avataren skal lastes:

- Dersom Avataren har proppen `name`, vil initialene vises.
- Dersom Avataren ikke har `name`, vil default-avataren vises.
