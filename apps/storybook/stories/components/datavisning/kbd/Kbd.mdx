# Keyboard Key (Kbd)

Kbd brukes til å vise hvilken tast eller kombinasjon av taster på tastaturet som vil sette i gang en handling.

```jsx
import { Kbd } from "@kvib/react";
```

<DocsHeading light>Guideline</DocsHeading>

Alle snarveier skal prøve å passe tastene på brukerens tastatur.

- Alle single bokstaver A-Å skal ha stor bokstav.
- Alle taster uten bokstaver (enter, esc, shift, osv.) skal ha små bokstaver.
