# Toast

Toast bruker for å gi brukeren tilbakemelding etter en handling. Toast skal forsvinne enten når den blir lukket,
eller av selg selv etterhvert. Flere toasts kan være på skjermen samtidig.

## Ta i bruk

```jsx
import { Toast } from "@kvib/react";
```
