# Header

Header er et sammensatt komponent som skal brukes øverst på Kartverkets sider. Arbeidet med Header er ikke ferdigstilt - endringer må påregnes.

## Ta i bruk

```jsx
import { Header, HeaderProps } from "@kvib/react";
```

## Plassering av innhold

Header kan ha innhold til venstre, med plass i mellom, eller sentrert. Default er "space-between".

<Canvas of={HeaderStories.HeaderJustifyContent} />

## Header breakpoint

Header har et breakpoint for når elementene vil fjernes og bli erstattet av en menyknapp. Menyknappen aktiverer en dropdown som inneholder elementene som er fjernet.
Default breakpoint for Header er "sm". Prøv å endre størrelsen på vinduet for å se hvordan det fungerer.

<Canvas of={HeaderStories.HeaderBreakpoint} />

## Header med tilpasset dropdown meny

Defaultinnhold i dropdown menyen under headeren er samme children som headeren har. Dropdownen kan tilpasses slik du vil ha den ved hjelp av disse propsene: `showChildrenInMenu` og `dropdownMenuChildren`.
`showChildrenInMenu=false` fjerner alt standardinnhold fra dropdownen og `dropdownMenuChildren` setter innholdet i dropdownen.
