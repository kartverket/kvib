# Table

Table brukes til å organiserer og vise frem data på en effektiv måte. Den render et `<table>` element som standard.

## Ta i bruk

```jsx
import { Table, Thead, Tbody, Tfoot, Tr, Th, Td, TableCaption, TableContainer } from "@kvib/react";
```
