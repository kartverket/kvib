# Icon

Vi benytter ikoner som et virkemiddel for hjelpe brukere med å forstå handling eller kontekst. Ikoner kan være med på å
skille og fremheve innhold.
Det er også et fint navigasjonsverkøy som kan vise varsling av feil eller fortelle brukeren om de forlater nettsiden.
[Her er det noen guidelines for bruk av ikoner](https://www.intechnic.com/blog/icon-usability-best-ux-tips-and-design-guidelines/)

Kartverkets Designsystem tilbyr et ikonelement som viser og lar deg style ikoner fra Google sin Material Symbols Rounded.
Gå til [Material symbols rounded sine sider](https://fonts.google.com/icons?icon.set=Material+Symbols&icon.style=Rounded)
for å få en oversikt over ikoner du kan benytte.

**Importer `MaterialSymbol` dersom du trenger type til icon-proppen.**

## Ta i bruk

<Alert status="info">
  <AlertIcon />
  **Dersom ikoner ikke vises:** `import 'material-symbols';` i index.js.
</Alert>

```jsx
import { Icon, MaterialSymbol } from "@kvib/react";
```
