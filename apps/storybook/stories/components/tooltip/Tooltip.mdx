Et tooltip er en informativ melding som dukker opp når en bruker samhandler med et element.

## Ta i bruk

```jsx
import { Tooltip } from "@kvib/react";
```

## Bruk av egne Komponenter

Dersom Tooltip wrapper et funksjonelt komponent må komponentet ha en ref ved `forwardref`. Dette inkluderer også bruk av ikoner fra KVIB. Se eksempel:

<Source code={`const CustomCard = React.forwardRef(({ children, ...rest }, ref) => (
  <Box p="1">
    <Tag ref={ref} {...rest}>
      {children}
    </Tag>
  </Box>
));

const CustomToolTip = () => (

<Tooltip label="Hover me">
  <CustomCard>Tag Here</CustomCard>
</Tooltip>
);

render(<CustomToolTip />);`} dark />
