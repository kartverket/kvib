# Heading

Heading skal brukes til overskrifter. Heading komponerer Box så det er mulig å bruke alle style props og legge til responsive stiler.
Den render en `<h2>`som standard. Dersom du ønsker at overskriften skal avkortes etter et spesifikt antall linjer, kan du bruke `noOfLines`-propen.

## Ta i bruk

```jsx
import { Heading } from "@kvib/react";
```
