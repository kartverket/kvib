# SimpleGrid

SimpleGrid er en enkel komponent for å bygge opp layout. Den er basert på css grid og har støtte for responsivitet.

```jsx
import { SimpleGrid } from "@kvib/react";
```

**Egnet til:**

- Oppbygging av enkel sidelayout
- Gruppering av elementer (f.eks. knapper, cards)

**Uegnet til:**

- Komplekse layouter (bruk heller Grid)

<DocsStory
  title="Columns"
  description={
    <Text>
      <Code>columns</Code>-proppen styrer antall kolonner i gridet. Den kan enten være et tall eller et array med tall.
      Hvis den er et array vil gridet automatisk justere seg etter skjermstørrelse. For eksempel
      <Code>{`columns={[2, null, 3]}`}</Code> eller <Code>{`columns={{ sm: 2, md: 3 }}`}</Code>.

<DocsStory
  title="Auto-responsive grid"
  description={
    <Text>
      For å gjøre gridet responsive og justere seg automatisk uten å bruke <Code>columns</Code> kan man bruke
      <Code>minChildWidth</Code> til å spesifisere <Code>min-width</Code> på hvert element. Gridet vil da automatisk justere
      seg når elementene ikke får plass. Løsningen benytter seg av css grid <Code>auto-fit</Code> og <Code>
        minmax()
      </Code>.
    </Text>
  }
  story={<Canvas of={SimpleGridStories.SimpleGridResponsive} />}
