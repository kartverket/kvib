# Stack

Stack er en layout komponent brukt til 책 gruppere sammen elementer og legge til spacing mellom dem.

**VStack:** brukes til 책 stacke elementer i vertikal retning\
**HStack:** brukes til 책 stacke elementer i horisontal retning\
**Stack:** brukes til 책 stacke elementer i vertikal eller horisontal retning

```jsx
import { Stack, HStack, VStack } from "@kvib/react";
```
