# Stack

Stack er en layout komponent brukt til å gruppere sammen elementer og legge til spacing mellom dem.

**VStack:** brukes til å stacke elementer i vertikal retning\
**HStack:** brukes til å stacke elementer i horisontal retning\
**Stack:** brukes til å stacke elementer i vertikal eller horisontal retning

```jsx
import { Stack, HStack, VStack } from "@kvib/react";
```
<DocsContainer>

**Egnet til**

- Sette opp innholdet i skjemaer
- Gruppere Card

**Uegnet til**

- Lage tabeller. Bruk heller Table komponenten.

Stack og Flex er veldig like, med at begge distanserer Children like langt fra hverandre. Den største forskjellen er at Stack ikke fyller bredden til hele containeren, mens Flex fyller hele bredden til containeren.
