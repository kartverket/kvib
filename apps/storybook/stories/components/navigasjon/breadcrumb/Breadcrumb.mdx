# Breadcrumb

Breadcrumb er et navigasjonsmønster som hjelper brukerne å forstå hierarkiet på en nettside. Legg til `ìsCurrentPage`-proppen på `BreacrumbItem` brukeren står i.
Breadcrumb inneholder `Box` så det er mulig å bruke alle style props derfra.

## Ta i bruk

```jsx
import { Breadcrumb, BreadcrumbItem, BreadcrumbLink, BreadcrumbSeparator } from "@kvib/react";
```
