# Range Slider

RangeSlider er en slider med to dragbare knapper som brukes til å velge område med verdier.

## Ta i bruk

```jsx
import { RangeSlider, RangeSliderTrack, RangeSliderFilledTrack, RangeSliderThumb } from "@kvib/react";
```

## Få tak i siste valgte verdi på slider

For å få tak i siste valgte verdi fra slideren etter den har blitt oppdatert flere ganger, kan man bruke `onChangeEnd`.

```jsx
<RangeSlider aria-label={["min", "max"]} onChangeEnd={(val) => console.log(val)}>
  <RangeSliderTrack>
    <RangeSliderFilledTrack />
  </RangeSliderTrack>
  <RangeSliderThumb index={0} />
  <RangeSliderThumb index={1} />
</RangeSlider>
```
