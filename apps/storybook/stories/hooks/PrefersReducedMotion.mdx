# usePrefersReducedMotion

**usePrefersReducedMotion** er en tilpasset hook som brukes til å hjelpe med å oppdage brukerens bevegelsespreferanse.

```jsx
import { usePrefersReducedMotion } from "@kvib/react";
```

<Feedback component="usePrefersReducedMotion"/>

# Return value
**usePrefersReducedMotion** returnerer en boolsk verdi som indikerer om brukeren foretrekker redusert bevegelse.
