# useOutsideClick

**useOutsideClick** er en tilpasset hook som håndterer klikkhendelser utenfor et bestemt DOM-element, som for eksempel en **div**. En håndterer kalles når en klikk- eller trykkehendelse skjer utenfor det refererte elementet.

<Alert status="warning">
    <AlertIcon />
    <Box>
        <AlertDescription>
            Denne hooken er kompatibel med mus- og trykkehendelser.
        </AlertDescription>
    </Box>
</Alert>

```jsx
import { useOutsideClick } from "@kvib/react";
```
