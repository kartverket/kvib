# Icon

Vi benytter ikoner som et virkemiddel for hjelpe brukere med å forstå handling eller kontekst. Ikoner kan være med på å skille og fremheve innhold.
Det er også et fint navigasjonsverkøy som kan vise varsling av feil eller fortelle brukeren om de forlater nettsiden.

Kartverkets designsystem tilbyr et ikonelement som viser og lar deg style ikoner fra
[Material symbols rounded.](https://fonts.google.com/icons?icon.set=Material+Symbols&icon.style=Rounded)

## Ta i bruk

```jsx
import { Icon } from "@kvib/react";
```
