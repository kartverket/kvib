* Legg inn nye ikoner ved behov.
* Ikoner bør hovedsaklig være outlined, og hentes ut av Google Material
* Ikonets farge må ha god nok kontrast til bakgrunnen.
* Små og ekstra små flater (Dashboard, kart og mobil) skal ha ikonstørrelse 20, Tablet og desktop kan bruke størrelse 24

### Eksempelbruk

Dette ikonet har kun outline:

```markdown
<div>
<span class="material-symbols-outlined">home</span>
</div>
```

Output: 
<div>
<span class="material-symbols-outlined">home</span>
</div>

<hr/>

Dette ikonet har fyll:
```markdown
<div>
<span class="material-symbols-outlined icon-filled">home</span>
</div>
```

Output:
<div>
<span class="material-symbols-outlined icon-filled">home</span>
</div>
