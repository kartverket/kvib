Implementering av KVIB

Installasjon av KVIB-CSS​

KVIB kan implementeres på tre måter:

1. HTML link​

Importer KVIB HTML link:

 <link rel="stylesheet" href="https://kartverket.github.io/kvib/css/main.css">

2. CSS import​

Importer KVIB i index.css i prosjektet: 

@import url("https://kartverket.github.io/kvib/css/main.css");

3. Installasjon av KVIB GitHub Package​

GitHub​

  1. Opprett et Personal Access Token (PAT). Se Github dokumentasjon for å opprette PAT open_in_new
  2. Autoriser PAT for bruk med organisasjonen. Se Github dokumentasjon for PAT autentisering open_in_new

npm​

  1. Opprett en .npmrc fil i prosjektmappen og legg til følgende:
@kartverket:registry=https://npm.pkg.github.com
  1. I prosjektmappen som inneholder .npmrc filen, logg på npm med GitHub brukernavn og PAT ved å bruke følgende kommando:
npm login --scope=@kartverket --registry=https://npm.pkg.github.com
  1. Kjør npm install @kartverket/kvib og biblioteket skal nå være klar til bruk.
npm install @kartverket/kvib
  1. Importer eller legg til link til biblioteket i CSS eller HTML filen i prosjektet.
  • I HTML fil:
<link rel="stylesheet" href="PATH/node_modules/@kartverket/kvib/main.css">
  • I CSS fil:
@import 'PATH/node_modules/@kartverket/kvib/main.css';
